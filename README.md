# 🏥 植物人大戰殭屍 🧟

> 「當醫院被入侵，只有躺著的勇士能拯救世界...」

一款黑色幽默風格的塔防遊戲，玩家放置躺在病床上的「植物人」來抵禦殭屍入侵醫院。

## 如何遊玩

1. 用瀏覽器開啟 `index.html`
2. 點擊「開始遊戲」
3. 選擇下方角色卡片
4. 點擊網格放置植物人
5. 撐過 7 波殭屍（含殭屍王）即可獲勝！

## 遊戲機制

### 基本規則
- **網格大小**：6 行 × 12 列
- **初始點數**：200 💰
- **總波數**：7 波（最後一波出殭屍王）
- **勝利條件**：撐過所有波次
- **失敗條件**：任何殭屍到達最左側

### 核心系統

| 系統 | 說明 |
|------|------|
| 🪏 **鏟子** | 移除已放置的植物人，回收 50% 成本 |
| ⏱️ **冷卻** | 每種植物人放置後需等待冷卻才能再次使用 |
| ❤️ **血條** | 植物人和殭屍都有血條，血量低時變色警示 |

## 策略建議

### 開局佈陣
1. 先放 2-3 個「氧氣罩手」🛏️😷 建立經濟
2. 每行至少放 1 個「輪椅擋板」🦽🛡️ 當肉盾
3. 肉盾後面放「點滴槍手」🛏️💉 輸出

### 進階技巧
- 「急救站」🛏️❤️ 放在肉盾旁邊可以持續補血
- 「麻醉師」🛏️💤 + 「冷凍櫃」🛏️🧊 組合可以讓殭屍幾乎不動
- 「X光射手」🛏️☢️ 穿透攻擊適合殭屍密集時使用
- BOSS 來襲時準備「氧氣瓶炸彈」🛏️💣 快速削血

### 經濟管理
- 善用鏟子回收不需要的植物人
- 氧氣罩手越早放越划算
- 擊殺殭屍也會獲得點數

## 響應式設計 (RWD)

遊戲支援各種裝置，手機、平板、桌面都能玩！

### 螢幕適配

| 裝置 | 螢幕寬度 | 網格大小 | 格子尺寸 |
|------|----------|----------|----------|
| 🖥️ 桌面 | ≥900px | 12×6 | 60px |
| 📱 平板 | 600-899px | 10×5 | 50px |
| 📱 手機 | 400-599px | 9-10×5 | 40-45px |
| 📱 小手機 | <400px | 8-9×5 | 35-40px |

### 手機操作提示

- 📱 **橫向遊玩**體驗最佳
- 👆 直接點擊選卡和放置
- 👈👉 可左右滑動查看完整地圖
- 🔒 已禁用雙擊縮放，避免誤觸

### 自動調整功能

- 網格大小根據螢幕動態調整
- 所有元素（血條、子彈、特效）等比例縮放
- 小螢幕自動隱藏說明區塊，節省空間
- 直向使用時顯示「橫向使用體驗更佳」提示
